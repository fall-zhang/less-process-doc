## 快捷键参考

> 一般再次使用快捷键会移除当前文字效果

### 文字效果

- 加粗 `ctrl` + `b`
- 斜体 `ctrl` + `i`
- 下划线 `ctrl` + `u`
- 行内代码块 `ctrl` + `e`
- 删除线 `ctrl` + `shift` + `s`

### 标题效果

- 一级标题 `ctrl` + `alt` + `1`
- 二级标题 `ctrl` + `alt` + `2`
- 三级标题 `ctrl` + `alt` + `3`
- ...

### 常用操作

- 复制 `ctrl` + `c`
- 粘贴 `ctrl` + `v`
- 复制为纯文本 `ctrl` + `shift` + `c`
- 粘贴为纯文本 `ctrl` + `shift` + `v`
