## 什么是 Markdown

markdown 是轻量标记文本，使用简单的标记即可实现对文档的编辑

## 为什么使用 markdown

### 便于查找

因为笔记便于查找
如果是语音，视频记录的信息，那怕十倍速播放，也要查好久
并且对于和机器的交互，你无法凭空输入一段视频，或者是图片，只能是文字描述
对于查找效率，文字最高，音频次之（转文字后，可以根据文字搜索），图片次之，视频更次。

### 空间据用小

一首无损音乐，10M，一张图片，100k，一段视频 10M

1MB，大致等于一本书，256,000 字，1G，等于一千本书，
5G，等于五千本书如果线下模式，用户想用多少用多少

### 格式应用广泛

在你目之所及的许多地方都有 markdown 的身影

- 微信公众号的文章
- 知乎的文章编辑
- SSG 静态网站生成
- 大多数应用的官方文档
- github 的 README

markdown 拥有多种实现，是互联网基建，可以放心使用，并且有统一的规范 [common-mark](https://commonmark.org/)

该应用使用通用的 markdown 解析，保证其它应用也能打开本地所有文件。

### 自定义样式

可以一键更改生成的样式

代码高亮

拥有超多扩展工具

### 网页静态部署

markdown 可以被解析为 HTML，加上 CSS 即可成为静态网页，世界上任何位置都可以通过网络进行访问你的网页。
